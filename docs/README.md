# 기능
## Car
- move
```
0~9숫자를 무작위로 뽑음
4이상이 나오면 location을증가
이름 : -(location길이)를 출력
```

## CarRacingSimulation 
- simulate
```
car 객체들을 round 수만큼 move() 메소드를 실행시킴
```


- getWinners()
```
car 객체들을 location 순으로 정렬후 첫번째(가장큰) location을 maxLocation에 저장
maxLocation인 car 객체들의 이름을 winnerCarNames 리스트에 저장후 출력
```
